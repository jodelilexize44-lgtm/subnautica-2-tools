Subnautica 2 Community Reports and Discussion Rooms

A place to organize player evidence and future discussion before it becomes public guide data: screenshots, crash details, co-op tests, save behavior, resource names, blueprint unlocks, map notes, build changes, and player questions.

Community Rooms

The site is static today, so these are topic rooms and report lanes rather than live forum threads. A real posting system should only be added after moderation, spam controls, account policy, and privacy updates are ready.

Help deskCrash, install, Game Pass, cloud play, and settings questions that need structured troubleshooting.
Co-op labHost save, guest progress, invite flow, reconnect, cross-store, and cross-platform tests.
Discovery boardResources, blueprints, routes, biomes, creatures, vehicles, base modules, and spoiler labels.
Build showcaseBase layout reports, purpose, platform, build date, screenshots, and material notes.
Gallery deskBeautiful screenshots, high-end PC captures, ambience observations, and official media references.
Patch watchPlayer retests after updates, changed behavior, fixed issues, and newly broken workflows.

Future Community Features

This is currently a static site, so it cannot host a real forum without adding a backend or an embedded third-party discussion system. The right path is to start with structured submissions, then add discussion only when moderation and spam control are ready.

Screenshot galleryCurated player screenshots with permission, spoiler labels, platform, and build date.
Build report boardPlayer-submitted performance, crash, co-op, save, and handheld reports.
Forum laterAdd real posting only after moderation, anti-spam, account policy, reporting, and privacy updates are designed.

What Will Not Be Published

  • Uncredited screenshots copied from other players without permission.
  • Fake maps, fake resource names, fake blueprint unlocks, or invented room/material costs.
  • Cheat, exploit, piracy, account-selling, or misleading download links.
  • One-line bug claims without platform, build, or reproduction details.
