# Subnautica 2 Launch Data Checklist

Use this file during the first 48 hours after Early Access launch. Fill only what is verified. Leave unknowns blank instead of guessing.

## Status Key

- `Confirmed`: directly tested or official.
- `Community report`: repeated player reports, not tested by us.
- `Unverified`: single report or not enough evidence.
- `Changed`: patched or contradicted by newer data.

## 9. Daily Review Checklist

Morning:

- Check GSC sitemap status.
- Check GSC Pages report.
- Check GSC Performance impressions.
- Check Steam discussions for repeated problems.
- Check official channels for patches.

Evening:

- Update `guides/updates.html`.
- Expand the page with the most useful new verified data.
- Request indexing for materially updated pages.
- Record all changes in this checklist or commit messages.
