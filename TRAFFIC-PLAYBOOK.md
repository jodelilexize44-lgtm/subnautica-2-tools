# Subnautica 2 Tools Traffic Playbook

This playbook is for ethical launch-week traffic. The goal is to answer real player questions and link to useful pages only when the link genuinely helps.

## Rules

1. Do not spam.
2. Do not pretend to be an unrelated user.
3. Disclose lightly when linking: "I made a small tool/checklist for this."
4. Prefer replying to existing questions over starting promotional threads.
5. Never post unverified claims as facts.
6. Stop posting in a thread if moderators or users object.
7. Keep links deep and relevant. Do not always link the homepage.

## Thread Types and Response Templates

### 1. Release Time Questions

Trigger examples:

- "When does it unlock?"
- "Where's EA release?"
- "What time is Subnautica 2 out?"

Template:

```text
Steam lists May 14, 2026. The exact unlock minute can vary by store, so I would still check the store countdown close to launch.

I made a small launch checklist/countdown page here if it helps:
https://jodelilexize44-lgtm.github.io/subnautica-2-tools/tools/release-time.html
```

Do not post if someone has already answered with a better official source unless adding the checklist is genuinely useful.

### 2. System Requirements / "Can I Run It?"

Trigger examples:

- "Can GTX 1650 run it?"
- "What specs do I need?"
- "Will it run on my laptop?"

Template:

```text
The Steam minimum is i5-8400 / Ryzen 5 2600, 12 GB RAM, GTX 1660 6GB / RX 5500 XT 6GB, and 50 GB storage. Recommended is much higher: i7-13700 / Ryzen 7 7700X and RTX 3070 / RX 6700 XT.

I put those into a quick checker here:
https://jodelilexize44-lgtm.github.io/subnautica-2-tools/tools/system-requirements.html
```

If the user asks about a specific GPU, answer directly before linking.

### 3. Co-op Questions

Trigger examples:

- "How many players?"
- "Does co-op save progress?"
- "Can Steam and Xbox play together?"

Template:

```text
The store page says it supports 4-player co-op and lists cross-platform multiplayer, but save ownership and reconnect behavior still need launch testing.

I'm tracking the confirmed vs unconfirmed co-op details here:
https://jodelilexize44-lgtm.github.io/subnautica-2-tools/guides/co-op.html
```

Update the page before linking if we have new verified co-op data.

### 4. Crash / Low FPS / Black Screen

Trigger examples:

- "Game crashes on startup"
- "Black screen"
- "Low FPS"
- "Won't launch"

Template:

```text
First safe checks: update GPU drivers, verify game files, restart, and disable overlays one by one. If you can share platform + GPU + error message, that helps narrow it down.

I'm collecting confirmed launch issues and workarounds here:
https://jodelilexize44-lgtm.github.io/subnautica-2-tools/guides/crash-fix.html
```

Do not invent fixes. Ask for platform and hardware details.

### 5. Game Pass / Xbox Questions

Trigger examples:

- "Is it on Game Pass?"
- "Is PC Game Pass live?"
- "Can Game Pass play with Steam?"

Template:

```text
Steam is confirmed as the visible Early Access listing. For Game Pass/Xbox, I would verify against the Xbox/Microsoft Store page at launch because store timing can differ.

I'm tracking the Game Pass and cross-store status here:
https://jodelilexize44-lgtm.github.io/subnautica-2-tools/guides/game-pass.html
```

Only state "yes" or "no" after official store confirmation.

## What Not To Do

- Do not paste the same message everywhere.
- Do not reply to old or unrelated threads just to place a link.
- Do not use link shorteners.
- Do not argue with users who dislike self-links.
- Do not claim "official" or "complete".
- Do not post affiliate links until the site has real value and disclosure.

9. Threads asking blueprint/resource questions.

## How To Judge If Outreach Is Working

Early signals:

- Search Console impressions start appearing for linked page topics.
- GitHub Pages traffic shows page requests rising.
- Players reply with more specific questions.
- Threads organically mention or reuse the tool.

Bad signals:

- Downvotes, moderator warnings, or replies calling it spam.
- No clicks and no replies after repeated posting.
- Users bounce because pages are too thin.

If bad signals appear, stop posting links and improve the page first.
