Safe Handheld Setup

  1. Check the listed PC requirements before assuming handheld performance.
  2. Start with a conservative preset from the best settings guide.
  3. Run a short solo test before co-op, large bases, or deep routes.
  4. Record battery mode or TDP mode because handheld performance can change heavily by power profile.
  5. After a patch, retest before treating old reports as current.
