Subnautica 2 Beginner Guide

A conservative guide for the first two hours. Detailed routes are added only after verified playtesting.

First Priorities

  1. Stabilize oxygen, food, water, and safe return paths before chasing distant signals.
  2. Mark early resource spots with beacons or notes.
  3. Build a compact storage routine before exploring deeper.
  4. Scan everything that looks usable, then sort blueprints by survival value.

Good Habits

  • Return with spare oxygen instead of pushing every dive to the limit.
  • Keep one container for base parts and one for crafting materials.
  • In co-op, split scouting and collecting instead of four players chasing the same marker.
  • Log any bugged interaction with location, item, and patch version.

Official Context

Subnautica 2 sends players to a new alien world, with the story starting around the CICADA facility and a group of Pioneers. The game is still Early Access, so this guide avoids exact route claims until they are checked against the current build.
