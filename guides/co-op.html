Subnautica 2 Co-op Guide

Steam lists online co-op and cross-platform multiplayer, while Xbox store pages list cross-platform co-op capabilities. This guide separates store facts from save ownership, invite flow, and reconnect behavior that need current-build testing.

Confirmed

  • Subnautica 2 is described as playable alone or with friends in 4-player co-op.
  • Steam feature tags list single-player, online co-op, and cross-platform multiplayer.
  • Xbox store pages list online co-op and cross-platform co-op capabilities for Subnautica 2 Game Preview.
  • Broadband internet is listed in the official system requirements.

Recommended Group Setup

  • Let the most stable PC or console host the first shared world.
  • Before a long session, run a 15-minute test world and reconnect every player once.
  • Keep a note with platform, build version, host name, and any disconnect messages.
  • Assign roles: host/base keeper, scout, collector, scanner, route marker.
  • Do not assume cross-store behavior until your exact group tests it.

Co-op FAQ

How many players can play together?

Public store copy describes 4-player co-op. Current-build testing should confirm how invites, hosting, and reconnects work in practice.

Who owns the co-op save?

This is not confirmed here yet. We will test whether the host owns the save and whether guest progress persists after reconnecting.

Can Steam and Game Pass players play together?

Store pages list cross-platform multiplayer or co-op features, but this guide still needs current-build testing for the exact invite flow between Steam, PC Game Pass, and Xbox players.

What should we test before a long co-op world?

Test invite flow, reconnects, guest progress, blueprint sync, and whether everyone sees the same build version. A short test world can save a lot of frustration later.

Should my group wait for patches?

If your first session is important, check the updates page and known issues before starting a long co-op world.
