Subnautica 2 Crash Fixes

A live tracker for crashes, black screens, low FPS, and startup issues. Workarounds are marked as official, tested, or community reported instead of being treated as universal fixes.

Before You Troubleshoot

  • Confirm your PC meets the listed system requirements.
  • Update GPU drivers before judging launch performance.
  • Verify game files after a failed install or interrupted patch.
  • Disable overlays one at a time if the game fails to launch.
  • Write down platform, CPU, GPU, RAM, driver version, and exact error text before trying many fixes.

Safe Fix Order

  1. Restart the PC or console and try one clean launch.
  2. Check system requirements, especially RAM, GPU class, and free storage.
  3. Update GPU drivers if you are on PC.
  4. Verify game files or repair the install through your store.
  5. Disable overlays one at a time: Steam/Xbox overlay, Discord, capture tools, FPS counters.
  6. Lower settings and test a short session before changing multiple options.
  7. Only reinstall after the safer checks fail.

Exact Startup Error

If you see a warning like "Your game might not have exited correctly the last time you played" or a message telling you to verify minimum requirements, update graphics drivers, reset settings, or lower settings, treat it as a startup stability report rather than proof of one single cause.

StepWhy
Check your PC against the listed requirements.The warning itself points to minimum requirements as a possible cause.
Update GPU drivers, then restart before launching again.Driver state is one of the safest high-impact checks.
Verify game files through Steam, Xbox app, or the store you used.Interrupted patches can produce launch crashes.
Start from conservative settings using the best settings guide.Lowering settings can reduce startup pressure on below-minimum or borderline PCs.
Record platform, CPU, GPU, RAM, driver version, and exact wording.Detailed repeated reports are what turn a monitoring item into a useful confirmed workaround.

Crash Fix FAQ

Should I reinstall immediately?

Not first. Verify files and check for a known issue before reinstalling, because launch bugs can be patch-related rather than install-related.

Do overlays cause crashes?

They can in some games, but this page will only list overlay-specific fixes after repeated reports or testing support them.

Where will confirmed fixes appear?

Confirmed workarounds will appear in the Known Issues table and on the updates page.

Should I use launch commands from random comments?

Be careful. This page does not recommend launch commands, registry edits, or file deletion steps unless they are official, directly tested, or repeatedly reported with enough detail.

What if I see the "game might not have exited correctly" message?

Follow the exact startup error section above. Start with requirements, drivers, file verification, and conservative settings before trying risky fixes.
