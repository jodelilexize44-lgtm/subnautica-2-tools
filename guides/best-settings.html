Subnautica 2 Best Settings

Use these conservative Early Access settings as a starting point, not a benchmark promise. Subnautica 2 performance can change with patches, drivers, co-op sessions, and large-base scenes.

First run the system requirements checker. Then pick the closest row below and change one setting at a time so you know what helped.

Safe Test Routine

  1. Update GPU drivers and reboot before judging performance.
  2. Run a short solo session with a conservative preset.
  3. Change one setting at a time and write down what changed.
  4. Test a second short session after a patch before committing to long co-op play.
  5. If the game crashes, return to the safe troubleshooting order.

Best Settings FAQ

Is this a benchmark?

No. It is a conservative starting guide based on the listed requirements and safe testing habits. Real FPS needs current-build testing.

Should I use High settings on recommended hardware?

Start around Medium to High, then adjust after a short test. Early Access patches can change performance.

What should below-minimum PCs do?

Wait for current-build reports before buying for that PC. If you already own it, start at the lowest settings and treat stability as the first goal.

No FPS promises

This page avoids fake benchmark numbers. Use it as a starting point, then log current-build results.
